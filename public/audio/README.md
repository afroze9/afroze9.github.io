# XMB Audio Files

Place the following audio files in this directory:

## Required Files

1. **background.mp3** - Ambient background music (looping)
   - Should be calm, atmospheric music similar to PS3/PS4/PS5 menu music
   - Recommended: 2-4 minutes, seamless loop
   - Keep file size reasonable (< 2MB for web)

2. **navigate.mp3** - Navigation tick sound
   - Short click/tick sound when moving between items
   - Duration: ~50-100ms
   - Should be subtle and non-intrusive

3. **select.mp3** - Selection confirmation sound
   - Played when pressing Enter to select an item
   - Duration: ~100-200ms
   - Should feel satisfying and confirmatory

4. **back.mp3** - Back/cancel sound
   - Played when pressing Escape to go back
   - Duration: ~100-150ms
   - Should be softer than select sound

## Royalty-Free Sources

- [Freesound.org](https://freesound.org) - Search for "UI click", "menu select", "ambient music"
- [Pixabay](https://pixabay.com/sound-effects/) - Free sound effects
- [Zapsplat](https://www.zapsplat.com) - UI sounds category

## Tips

- Keep sound effects short and snappy
- Use MP3 format for best browser compatibility
- Normalize audio levels so they're consistent
- Background music should be quieter than UI sounds
